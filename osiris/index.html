Working Features:

    Alpha 3
  • Escapes the SandBox
  • Patches AMFI (CodeSign)
  • Remounts ROOT FS as R/W on iOS 11.2.x
  • Disables the iOS Updates and removes already downloaded ones
  • Spawns Dropbear for SSH (remote access from PC) on port 22 (Use iPhoneTunnel for tunneling)
  • Fixed the signing issue and uicache now works thanks to @IBSparkes who pointed me to the correct debugging tools
  • Fixed the File Descriptor Leak in multi_path thanks to luckycat889 on http://newosxbook.com/forum/

What needs to be implemented
